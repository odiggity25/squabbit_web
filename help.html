Common issues

  • A course has missing or incorrect data (missing tee, incorrect slope, etc.)

    You can edit the course information yourself including adding new tees and changing the hole S.I.

  • I can't find my course
    You can add new courses from the courses tab.
  • I forgot my password
    Tap the login button and you will see a forgot password button.
  • I never received a password reset email
    Check your spam folder, in almost all cases the email will be there.
  • There are duplicate users in my tournament/league/club
    Fixing duplicate users
  • How are handicaps calculated?
    How handicaps work in Squabbit
  • How can I make another player an admin?
    In your tournament, league or club, tap the gear icon -> 'Players' and then tap cell in the Role column for the player to make them an admin.
  • How can I have different teams in each round of the tournament?
    You can add the same format multiple times to your tournament and you can choose which tournament rounds each format applies to. So just add the format multiple times and for each time you add it, only enable it for a single round. Then you can create different teams in each format.
  • How can I see a net and gross leaderboard?
    You can add the same format multiple times to your tournament. So just add a Strokeplay format twice, once with handicaps enabled and once with handicaps disabled.
