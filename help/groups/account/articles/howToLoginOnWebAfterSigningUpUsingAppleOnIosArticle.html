How to login on web after signing up with Apple on iOS

At this time, Apple login is not supported on web. So if you initially signed up with Apple on the iOS version of Squabbit you will not be able to login on web without first adding a password to your account.

To add a password, first open Squabbit in a web browser. Then tap the Profile tab -> Profile settings -> Link email sign in. After adding a password you can log in using the web with your email and password.
